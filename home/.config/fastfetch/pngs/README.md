# Fastfetch Pngs
- Anime images for Fastfetch

### Usage
```
git clone https://github.com/Thunder-Blaze/FastfetchPngs.git --depth 1
```

- After the download you can copy the images to your own Fastfetch image directory

=> **Recommended to be used with [Tsukiyomi-Fetch](https://github.com/Thunder-Blaze/Tsukiyomi-Fetch)**
